# 🚀 SplitSmart Pro – Smart Expense & Financial Insights Platform

SplitSmart Pro is a modern web application that simplifies group expense management while providing powerful financial insights. It helps users track shared expenses, automatically split costs, and clearly understand who owes whom — all in a clean, intuitive interface.

---

## 📌 Problem Statement

Managing shared expenses among friends, roommates, or during trips is often confusing and error-prone. People struggle with:
- Tracking who paid what
- Manually splitting expenses
- Calculating balances
- Understanding spending patterns

---

## 💡 Solution

SplitSmart Pro provides a smart platform that:
- Tracks group expenses
- Automatically splits costs
- Calculates balances instantly
- Offers insights and visual analytics for better financial understanding

---

## 🔥 Key Features

### 🧾 Expense Management
- Create groups (e.g., trips, roommates)
- Add and track expenses
- Categorize expenses (Food, Travel, Rent, etc.)

### 🧮 Smart Expense Splitting
- Automatically calculates each member’s share
- Displays clear results:
  - “You owe ₹300”
  - “You are owed ₹500”

### 📊 Visual Analytics
- Pie chart for category-wise spending
- Bar chart for member-wise contributions

### 🧠 Smart Insights
- Identify highest spending categories
- Compare contributions between members
- Understand spending behavior

### 💰 Settlement System
- View who owes whom
- Simple “Settle Up” functionality

### 📈 Activity Tracking
- Timeline of recent transactions and activities

## 🚀 How It Works

1. User creates a group  
2. Adds expenses with details  
3. System automatically splits the amount  
4. Displays balance summary  
5. Users can settle payments easily
